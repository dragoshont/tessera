# Tutorials

Tutorials are **lessons**. Each one takes you through a small, complete example, step by
step, with a result you can see. They are for **learning** — when Tessera is new to you,
start here.

> A tutorial holds your hand. You do not need to understand everything yet; you will
> *do* it, and understanding follows. When you later want to perform a specific task,
> move to the [how-to guides](../how-to/README.md).

## Lessons

| Lesson | You will… |
|---|---|
| [Your first brokered call](01-your-first-brokered-call.md) | Run the broker on your own machine, write one rule, and watch Tessera make a real decision — from zero, with no cloud account. |

## What you need

- The .NET 10 SDK (to build and run the broker from source).
- A terminal.
- About fifteen minutes.

No cloud account, no secrets, and no network access to any provider are needed for the
first lesson. It runs entirely on your machine.
